Basic Pipe Example

The powerTopipe takes the base as the input and outputs the result of raising the base to the given exponent. The exponent is specified as a parameter to the pipe.

  2^3 = {{ 2 | powerTo:3 }} // -> 2^3 = 8

2is the input to the pipe (the base)

3is the exponent specified as the parameter of the pipe

The expression above results in the following:

2^3 = {{ 2 | powerTo:3 }}
